                <h1>千年伞韵 —— 油纸伞起源与发展</h1>
                <p>油纸伞承载着千年的东方美学与匠心。黔北赤水油纸伞因其独特的地理环境与精湛的竹雕竹编工艺，已荣列贵州省级非物质文化遗产名录，成为西南文化的一颗璀璨明珠。</p>
            </div>

            <h2 class="section-title">1. 起源传说</h2>
            <div class="content-card">
                <p>民间传说，油纸伞的创造者为我国木匠鼻祖<strong>鲁班</strong>的妻子<strong>云氏</strong>。相传云氏因见丈夫长年在外劳作，常遭雨淋，便萌发了“收拢如棍，张开如盖”的巧思，她将竹子劈成细条、蒙上兽皮，这便形成了世界上第一把伞的雏形。后人将其改良为纸张并涂刷桐油，演变为流传千年的油纸伞。</p>
            </div>

            <h2 class="section-title">2. 朝代发展时间线</h2>
            <div class="content-card">
                <div class="timeline">
                    <div class="timeline-item"><strong>先秦时期（雏形现世）：</strong> 主要是丝帛制成的“帷盖”，多为王公贵族遮阳彰显身份的礼仪道具。</div>
                    <div class="timeline-item"><strong>汉代时期（纸伞诞生）：</strong> 东汉蔡伦改良造纸术后，开始出现以纸张蒙上伞骨、并涂抹植物油防水的真正“油纸伞”。</div>
                    <div class="timeline-item"><strong>唐宋时期（全民普及）：</strong> 宋代造纸业兴旺，绿油纸伞在民间大为普及，文人墨客开始在伞面题诗作画，使其具备了极高的美学价值。</div>
                    <div class="timeline-item"><strong>明清时期（工艺成熟）：</strong> 制作工序走向精细化，形成了“七十二道工序”的严格标准，各大流派在全国百花齐放。</div>
                    <div class="timeline-item"><strong>近现代（赤水传承）：</strong> 贵州赤水油纸伞吸纳了黔北竹艺的精华。2026年的今天，它不仅是防雨工具，更作为文创非遗，走出大山，走向世界。</div>
                </div>
            </div>

            <h2 class="section-title">3. 地域油纸伞区分</h2>
            <div class="content-card">
                <div class="geo-grid">
                    <div class="geo-card">
                        <h3>江南油纸伞</h3>
                        <p>以秀丽、轻巧著称。伞面多以水墨江南、仕女花鸟为主，极具书卷气和浪漫诗意。</p>
                    </div>
                    <div class="geo-card">
                        <h3>川渝分水油纸伞</h3>
                        <p>骨架坚固，伞面厚实，耐用性极强，抗风防水性能在西南多雨山地表现优异。</p>
                    </div>
                    <div class="geo-card">
                        <h3>黔北赤水竹骨油纸伞</h3>
                        <p>精选赤水本地优质楠竹，伞骨处理融入了防霉防蛀的秘方，整体呈现出一种自然、古朴且极其坚韧的少数民族文化融合特征。</p>
                    </div>
                </div>
            </div>

            <h2 class="section-title">4. 文化寓意</h2>
            <div class="content-card">
                <p>油纸伞在传统文化中拥有深厚的象征意义：</p>
                <ul>
                    <li><strong>吉祥多子：</strong>“油纸”与“有子”谐音；繁体字“傘”字形由多个“人”字组成，寓意多子多福、多子多孙。</li>
                    <li><strong>婚嫁习俗：</strong>在南方许多地区的传统婚礼中，新娘下轿时媒婆要撑开一把红色油纸伞遮下，寓意避邪趋吉、婚姻圆满。</li>
                    <li><strong>经典意象：</strong>从戴望舒《雨巷》中那个丁香般的姑娘，到《白蛇传》中许仙与白娘子的西湖断桥借伞，油纸伞早已成为中国古典浪漫不可或缺的视觉图腾。</li>
                </ul>

            <h2 class="section-title craft-title">1. 原料选材</h2>
            <div class="content-card">
                <p>赤水油纸伞的坚韧长寿，源于对大自然近乎苛刻的选材：</p>
                <ul>
                    <li><strong>3年以上老楠竹：</strong>必须选用赤水深山中生长3-5年、经受过风霜的成年楠竹，其纤维密度高、韧性极佳，能保证骨架历经万次开合而不折断。</li>
                    <li><strong>高韧性皮纸（棉纸）：</strong>伞面采用特制的手工棉纸，拉力强、遇水不易破烂，吸墨吸油性能卓越。</li>
                    <li><strong>天然熟桐油：</strong>这是赋予油纸伞防水灵魂的关键。将天然桐树果实榨出的生桐油经过熬制变成熟桐油，涂刷后会形成一层坚硬透亮的保护膜。</li>
                </ul>
            </div>

            <h2 class="section-title craft-title">2. 五大工序分步详解</h2>
            <div class="content-card">
                <div class="step-list">
                    <div class="step-item">
                        <div class="step-num">1</div>
                        <div class="step-detail">
                            <h4>制伞骨（骨架挺拔之源）</h4>
                            <p>工艺包含：砍竹、泡水去糖（防止日后生虫发霉）、刮去绿色表皮、精准破竹削成粗细一致的细丝。随后进行复杂的打孔与开槽，确保伞骨错落有致。</p>
                        </div>
                    </div>
                    <div class="step-item">
                        <div class="step-num">2</div>
                        <div class="step-detail">
                            <h4>组装伞架（精巧结构成型）</h4>
                            <p>将长伞骨与短伞骨用坚韧的棉线一一穿绕串联，固定在伞杆与伞托上。这个过程需要匠人极度匀称的手劲，保证伞架能够平稳开合、受力均匀。</p>
                        </div>
                    </div>
                    <div class="step-item">
                        <div class="step-num">3</div>
                        <div class="step-detail">
                            <h4>裱糊伞面（薄如蝉翼的贴合）</h4>
                            <p>把高韧性的手工棉纸按照伞架的扇形切块。匠人一边转动伞架，一边将棉纸多层重叠裱糊在骨架上。浆糊要涂得薄而均匀，裱好后置于通风处自然阴干。</p>
                        </div>
                    </div>
                    <div class="step-item">
                        <div class="step-num">4</div>
                        <div class="step-detail">
                            <h4>手绘纹样（伞面美学升华）</h4>
                            <p>阴干后的白纸伞面就成了匠人的画布。赤水艺人手持画笔，在带有弧度的伞面上手工绘制山水、花鸟或富有黔北少数民族特色的吉祥图腾，赋予伞生命力。</p>
                        </div>
                    </div>
                    <div class="step-item">
                        <div class="step-num">5</div>
                        <div class="step-detail">
                            <h4>刷桐油（防水耐用的蜕变）</h4>
                            <p>在画好的伞面上反复均匀地涂刷多层熬制好的熟桐油。刷完油的伞面会由白变透明，展现出迷人的光泽，这也是油纸伞能够真正防雨、越用越坚韧的终极秘诀。</p>
                        </div>
                    </div>
                </div>

        // ------------------ 2. 面向对象的答题系统引擎 ------------------
        class NonFactorQuiz {
            constructor(questions, rootId, prefix) {
                this.questions = questions;
                this.root = document.getElementById(rootId);
                this.prefix = prefix;
                this.currentIndex = 0;
                this.score = 0;
                this.hasAnswered = false;
                
                // 绑定DOM
                this.domProgress = document.getElementById(`${prefix}-progress`);
                this.domQuestion = document.getElementById(`${prefix}-question`);
                this.domOptions = document.getElementById(`${prefix}-options`);
                this.domFeedback = document.getElementById(`${prefix}-feedback`);
                this.domStatus = document.getElementById(`${prefix}-status`);
                this.domExplain = document.getElementById(`${prefix}-explain`);
                this.domNextBtn = document.getElementById(`${prefix}-next-btn`);
                this.domGameView = document.getElementById(`${prefix}-quiz-game`);
                this.domResultView = document.getElementById(`${prefix}-result`);
                this.domScoreSpan = document.getElementById(`${prefix}-score`);
                this.domEvalP = document.getElementById(`${prefix}-eval`);
            }

            init() {
                this.currentIndex = 0;
                this.score = 0;
                this.domGameView.style.display = 'block';
                this.domResultView.style.display = 'none';
                this.renderQuestion();
            }

            renderQuestion() {
                this.hasAnswered = false;
                this.domFeedback.classList.remove('show');
                this.domNextBtn.style.display = 'none';
                
                const currentData = this.questions[this.currentIndex];
                this.domProgress.innerText = `第 ${this.currentIndex + 1} / ${this.questions.length} 题`;
                this.domQuestion.innerText = `${this.currentIndex + 1}. ${currentData.q}`;
                
                this.domOptions.innerHTML = '';
                currentData.o.forEach((option, idx) => {
                    const btn = document.createElement('button');
                    btn.className = 'opt-btn';
                    btn.innerText = option;
                    btn.onclick = () => this.checkAnswer(idx, btn);
                    this.domOptions.appendChild(btn);
                });
            }

            checkAnswer(selectedIdx, clickedBtn) {
                if (this.hasAnswered) return;
                this.hasAnswered = true;
                
                const currentData = this.questions[this.currentIndex];
                const allButtons = this.domOptions.querySelectorAll('.opt-btn');
                
                // 禁用所有选项
                allButtons.forEach(btn => btn.disabled = true);
                
                if (selectedIdx === currentData.c) {
                    this.score++;
                    clickedBtn.classList.add('correct');
                    this.domStatus.className = 'feedback-status correct-text';
                    this.domStatus.innerText = '︎ 答对了！';
                } else {
                    clickedBtn.classList.add('wrong');
                    allButtons[currentData.c].classList.add('correct'); // 高亮正确答案
                    this.domStatus.className = 'feedback-status wrong-text';
                    this.domStatus.innerText = ' 答错了~';
                }
                
                this.domExplain.innerText = currentData.e;
                this.domFeedback.classList.add('show');
                this.domNextBtn.style.display = 'inline-block';
            }

            next() {
                this.currentIndex++;
                if (this.currentIndex < this.questions.length) {
                    this.renderQuestion();
                } else {
                    this.showResult();
                }
            }

            showResult() {
                this.domGameView.style.display = 'none';
                this.domResultView.style.display = 'block';
                this.domScoreSpan.innerText = this.score;
                
                let evalText = '';
                if (this.score === 15) {
                    evalText = '太不可思议了！你对非遗油纸伞文化简直是全知全能，恭喜获得【首席非遗传承官】称号！';
                } else if (this.score >= 10) {
                    evalText = '非常优秀！你已经基本掌握了油纸伞的核心常识，是个合格的【文化推广大使】！';
                } else {
                    evalText = '没关系，非遗手艺博大精深。再次阅读上面的科普图文与实拍视频，相信下次你一定能拿满分！';
                }
                this.domEvalP.innerText = evalText;
            }

            restart() {
                this.init();
            }
        }

        // ------------------ 3. 实例化两个板块的答题系统 ------------------
        let HistoryQuiz, CraftQuiz;
        window.onload = function() {
            HistoryQuiz = new NonFactorQuiz(historyQuestions, 'history-quiz-root', 'h');
            CraftQuiz = new NonFactorQuiz(craftQuestions, 'craft-quiz-root', 'c');
            
            HistoryQuiz.init();
            CraftQuiz.init();
        };
    </script>
